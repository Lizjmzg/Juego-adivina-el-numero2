# Adivina el número 
👩🏽‍💻 Página web creada con HTML5, CSS y JavaScript 👩🏽‍💻
## Introducción 📖
Este proyecto consistió en una práctica donde puse a prueba mis habilidades en tecnologías como HTML para dar estructura a la página, CSS para dar estilo a la página y JavaScript para dar dinamismo a la página.
## ¿Qué construí? 🧱
Se construyó una página web dinámica en donde el usuario tiene que adivinar el número secreto del 1 al 10. Una vez introducido el número, se le avisa al usuario si el número a adivinar es mayor o menor. Una vez acertado el número secreto, se le avisa al usuario en cuántos intentos lo logró.

## Tecnologías 👩🏽‍💻
Esta página web fue creada con:
+ HTML
+ CSS
+ JavaScript
